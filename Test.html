To open the product, first, open the windows command prompt cmd. Direct it to the folder: \Product\x64\Release within the form.

To open training mode type: "Keyword Object Detection.exe" train

To open detection mode type: "Keyword Object Detection.exe" find

For both options, you can add -s to show images along the way; this is not recommended for the training mode.

You can also use -ls to show just the last image.
